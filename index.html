Projects

EpiJS

EpiJS is a Node.JS package and library for epidemiological modelling of disease outbreaks using compartmental models. It's quite similar to EpiSpot, another of my projects.

View on GitHub

epispot

Epispot is a Python package created for epidemiologists. It helps model disease outbreaks in the speed of light with built in models, and allows you to create your own, along with allowing you to predict the spread.

View on PyPI

DiffPriv

DiffPriv is a Python package created for differential privacy. It has built in methods for different mechanisms in differential privacy, and currently has over 2K downloads on PyPI.
