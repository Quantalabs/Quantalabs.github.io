Projects

These are some of my most recent projects, including Leaf, my work in progress programming language.

Lamp

Lamp is a JavaScipt Canvas library which allows you to make graphics in the speed of light. Instead of writing 5 lines on code for just a triangle, built in functions and classes can help. You don't have to take 10 lines to setup the canvas, and do it all with one line.

View on GitHub

epispot

Epispot is a Python package created for epidemiologists. It helps model disease outbreaks in the speed of light with built in models, and allows you to create your own, along with allowing you to predict the spread.

View on PyPI

DiffPriv

DiffPriv is a Python package created for differential privacy. It has built in methods for different mechanisms in differential privacy, and currently has over 2K downloads on PyPI.
