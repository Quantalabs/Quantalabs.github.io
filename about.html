About me

I'm Quantalabs, a musician, developer, and hobbyist. I mainly work with python and JS, with some of my most recent projects including DiffPriv, a differential privacy package for python, and epispot, another python package for epidemiologists to model disease outbreaks, including COVID-19.

I'm currently working on epispot and a new project: Leaf. Leaf is a new programming language set to be all up and running by spring. This is a collaboration which I'm working on with quantum9innovation.

Currently Working on...

epispot

Epispot is a Python package created for epidemiologists. It helps model disease outbreaks in the speed of light with built in models, and allows you to create your own, along with allowing you to predict the spread.
